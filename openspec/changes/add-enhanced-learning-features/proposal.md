# Change: 增强学习功能模块

## Why
当前AI家教软件虽然具备基础的对话和文件上传功能，但缺乏系统化的学习管理工具和完善的对话历史管理。用户需要更智能的学习辅助功能，包括对话历史保存、收藏管理、知识管理、个性化学习计划、学习提醒和内容总结等，以提升学习效率和体验。

## What Changes
- **对话轮次管理和收藏**：实现每轮对话的独立保存和收藏功能，每个对话轮次配备星星收藏按钮
- **四模块界面重构**：将整体架构重构为四个主要模块：对话、收藏、知识库、智能学习助手
- **优化图片上传功能**：增强现有的文件上传模块，支持更多格式、批量上传、预览优化和智能分类
- **建立知识库系统**：新增知识库模块，支持目录组织、标签分类、搜索和收藏功能
- **添加学习助手功能**：新增学习助手模块，集成学习方法论（遗忘曲线、费马学习法），支持自定义学习计划和AI定时提醒
- **添加内容总结功能**：在对话界面添加智能总结按钮，支持关键词选择、多角度思考和可配置的总结详细程度

## Impact
- **新增规范**：
  - `favorites` - 收藏管理模块
  - `knowledge-base` - 知识库管理模块
  - `learning-assistant` - 学习助手模块
  - `content-summary` - 内容总结模块
- **修改规范**：
  - `file-upload` - 增强图片上传功能
  - `ai-chat` - 添加对话轮次管理和总结功能按钮
  - `user-interface` - 重构为四模块导航架构
- **代码影响**：
  - 重构主界面导航系统为四模块架构
  - 新增对话轮次数据模型和存储
  - 新增收藏管理界面和API
  - 新增知识库管理界面和API
  - 新增学习助手界面和逻辑
  - 新增内容总结功能
  - 扩展文件上传和对话界面功能
  - 增加用户数据存储和管理
